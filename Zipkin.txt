Download the JAR at https://zipkin.io/pages/quickstart.html

Google -> zipkin quick start
1. Go to Java approach - Minimum we need Java 8 installed on our machine
2. Get the URL to download the zipkin jar
3. Copy the JAR in to some folder
4. set RABBIT_URI=amqp://localhost - Tell Zipkin to listen to RabbitMQ Message Broker runnig at local machine
5. java -jar zipkin-server-2.12.3-exec.jar
6. Start the services and hit the RESTful URI
7. Zipkin dash board -> localhost:9411/zipkin/


Running Sequence
1. Eureka Naming Server
2. Zuul Api Gateway Server
3. Zipkin Distributed Tracing Server (Exercise on how to connect Zipkin to Eureka)
